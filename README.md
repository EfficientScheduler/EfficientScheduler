## 主文件选项

- ### `app`

  - `"package"` = `String`

    - `package`: 字符串，应用包名
      - `String`: 需为 powersave, balnace, performance, fast

- ### `on`

  - `on` = `String`

    - `on`: 不在[App 列表的](#app)
      - `String`: 需为 powersave, balnace, performance, fast

- ### `off`

  - `off` = `String`

    - `on`: 息屏时
      - `String`: 需为 powersave, balnace, performance, fast
